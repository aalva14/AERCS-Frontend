AERCS : Academic Event Recommender system for Computer Scientists
Information
AERCS is a visualization tool for communities of academic event in Computer Science. It is developed and hosted by the Chair of Information System , RWTH Aachen University, Germany. Currently, co-authorship network of conferences and series from DBLP dataset are visualized. Other data sets will be intergrated in the future.
Features
You can browse a list of conferences series and select a particular series to see the detail analysis and visualization of the community. Each series and conference has its own page which provides more detail information. Each author also has a page which list all conferences he/she has attended and a visualization of his/her research community. Users can also search conferences, series, or author by name.
Note
To reserve resource in order to serve multiple users, each user has a 10-minute session. When you do nothing within 10 minutes, then you may see nothing if you try to continue to interact with the graph. In this case, all you need to do is to refresh the whole page to create a new session.
